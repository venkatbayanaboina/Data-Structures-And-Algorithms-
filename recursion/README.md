# Recursion

Recursion involves a function calling itself to solve smaller instances of the same problem. Master base cases, recurrence relations, and the call stack.

## 🎯 Question Checklist

### Easy
- [ ] Print 1 to N
- [ ] Factorial of N
- [ ] Fibonacci Number
- [ ] Check if String is Palindrome

### Medium
- [ ] Power (x, n)
- [ ] Subsets / Power Set
- [ ] Generate Parentheses
- [ ] Tower of Hanoi
