# Heaps (Priority Queues)

Heaps are tree-based structures that satisfy the heap property (Min-Heap or Max-Heap). Useful for finding the top-K elements or dynamically tracking the median.

## 🎯 Question Checklist

### Easy
- [ ] Kth Largest Element in a Stream
- [ ] Last Stone Weight

### Medium
- [ ] K Closest Points to Origin
- [ ] Top K Frequent Elements
- [ ] Task Scheduler

### Hard
- [ ] Find Median from Data Stream
- [ ] Merge k Sorted Lists
- [ ] Smallest Range Covering Elements from K Lists
