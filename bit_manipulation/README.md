# Bit Manipulation

Bit manipulation involves working with individual bits using operators like AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), and bit shifts (`<<`, `>>`).

## 🎯 Question Checklist

### Easy
- [ ] Number of 1 Bits
- [ ] Counting Bits
- [ ] Reverse Bits
- [ ] Single Number
- [ ] Missing Number

### Medium
- [ ] Sum of Two Integers
- [ ] Subsets (Bitmask technique)
- [ ] Divide Two Integers
